Краткое описание

Кодинг-платформа — это веб-приложение для решения задач по программированию с автоматической проверкой решений и анализом ошибок с помощью LLM (больших языковых моделей).

Основные возможности:

  • Регистрация и авторизация пользователей
  • Решение задач с помощью встроенного редактора кода
  • Автоматическая проверка решений на тестовых случаях
  • Интеллектуальный анализ ошибок с помощью LLM
  • Управление задачами для преподавателей и администраторов
