О проекте Coding Platform

Что такое Coding Platform?

Coding Platform — это образовательная платформа для обучения программированию на языке Python. Система позволяет решать задачи различной сложности с автоматической проверкой решений и получать интеллектуальный анализ ошибок с использованием современных технологий искусственного интеллекта.

Основные возможности
  • Библиотека задач - доступ к задачам различной сложности и тематики
  • Автоматическая проверка - мгновенная обратная связь о вашем решении
  • Анализ ошибок с помощью ИИ - интеллектуальный помощник для поиска и исправления ошибок
  • Персональная история решений - отслеживание вашего прогресса
  • Фильтрация и поиск задач - быстрый поиск подходящих задач по теме и сложности
Роли пользователей
Роль Возможности
Пользователь Решение задач, просмотр своих решений, доступ к анализу ошибок
Учитель Все возможности пользователя + создание, редактирование и удаление задач
Администратор Все возможности учителя + управление пользователями и системой
Используемые технологии
  • Backend: Python, Flask
  • База данных: SQLite
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Анализ ошибок: OpenAI API
  • Выполнение кода: Изолированное выполнение пользовательского кода
